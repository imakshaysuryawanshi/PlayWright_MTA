# JavaScript Important Keywords

| Keyword | Description | Category |
|---|---|---|
| `var` | Declares a variable (function scope) | Variable Declaration |
| `let` | Declares a block-scoped local variable | Variable Declaration |
| `const` | Declares a block-scoped, read-only constant | Variable Declaration |
| `if` | Executes a block if a condition is true | Conditional |
| `else` | Executes a block if the `if` condition is false | Conditional |
| `switch` | Evaluates an expression and matches cases | Conditional |
| `case` | Specifies a branch in a `switch` statement | Conditional |
| `default` | Specifies the default branch in `switch` or exports | Conditional / Module |
| `for` | Creates a loop with 3 optional expressions | Loop |
| `while` | Creates a loop that executes while a condition is true | Loop |
| `do` | Creates a `do...while` loop | Loop |
| `break` | Terminates a loop or switch | Control Flow |
| `continue` | Skips the current iteration of a loop | Control Flow |
| `function` | Declares a function | Function |
| `return` | Exits a function and optionally returns a value | Function |
| `yield` | Pauses and resumes a generator function | Generator |
| `class` | Declares a class | Object-Oriented |
| `extends` | Creates a class that inherits from another class | Object-Oriented |
| `super` | Calls the parent class constructor/methods | Object-Oriented |
| `this` | Refers to the current object | Object-Oriented |
| `new` | Creates an instance of an object | Object-Oriented |
| `try` | Defines a block of code to test for errors | Error Handling |
| `catch` | Defines a block to handle errors | Error Handling |
| `finally` | Executes code after try/catch regardless of result | Error Handling |
| `throw` | Throws a custom error | Error Handling |
| `import` | Imports bindings from a module | Module |
| `export` | Exports functions/objects from a module | Module |
| `typeof` | Returns the type of a variable | Type/Operator |
| `instanceof` | Checks if an object is an instance of a constructor | Type/Operator |
| `in` | Checks if a property exists in an object | Type/Operator |
| `void` | Evaluates an expression and returns `undefined` | Type/Operator |
| `delete` | Deletes a property from an object | Type/Operator |
| `true` | Boolean true value | Literal |
| `false` | Boolean false value | Literal |
| `null` | Represents no value or empty value | Literal |
| `undefined` | Represents an uninitialized variable | Literal |
| `with` | Extends the scope chain for a statement (deprecated) | Scope |
| `debugger` | Invokes debugging functionality | Debugging |
| `async` | Declares an async function | Async/Wait |
| `await` | Waits for a Promise to resolve | Async/Wait |
| `static` | Declares a static method/property in a class | Object-Oriented |
| `get` | Defines a getter method | Object-Oriented |
| `set` | Defines a setter method | Object-Oriented |

> **Note:** `enum` is a reserved keyword in JavaScript but is not currently used.
